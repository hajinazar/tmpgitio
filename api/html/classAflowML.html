Detailed Description

This class submits a POSCAR to the Aflow ML server in order to get back machine learning results from it. It generates a separate thread and runs everything in the separate thread. submitPoscar() returns a request Id that can be used to keep track of the request that was sent. And a received() signal is emitted when the result from the machine learning calculation is obtained.
