void GlobalSearch::Bond::atomIndexRemoved ( size_t  atomInd)
inline

Indicate to this bond that the atom index, atomInd, has been removed, so any indices higher than this should be decremented. Because the index has been removed, all bonds to the removed atom should have already been deleted. This bond SHOULD NOT have one of its indices equal to atomInd.

Parameters

◆ swapIndices()

void GlobalSearch::Bond::swapIndices ( size_t  ind1,
size_t  ind2 
)
inline

This function is to be used when swapping atom indices in a molecule. If this bond contains ind1 as one of it's indices, it will swap it with ind2. If this bond contains ind2 as one of it's indices, it will swap it with ind1. This function does nothing if the bond does not contain ind1 or ind2.

Parameters
ind1The index (if it exists) to be swapped with ind2.
ind2The index (if it exists) to be swapped with ind1.
