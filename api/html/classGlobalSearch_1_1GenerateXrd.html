The primary function that is to be called in this class. This function takes a Structure object, converts it to a cif file, generates Xrd data from it, and stores it in results.

Parameters
sThe Structure whose XRD is to be obtained.
resultsThe resulting data (it is a vector of pairs of doubles). The first element in the pair is the 2theta value in degrees, and the second element in the pair is the intensity.
wavelengthThe wavelength of the x-ray in Angstroms.
peakwidthThe broadening of the peak at the base (in degrees).
numpointsThe number of 2theta points.
max2thetaThe max 2theta value in degrees.
Returns
True on success and false on failure.
