Installation

In the following, a set of guidelines based on successful compilation of the code on the mentioned systems are provided. It should be noted, however, that due to the extensive diversity of operating systems and architectures (consequently: libraries and compilers) it is challenging to provide a complete instruction for compilation of the code. The user might need to refer to the dedicated references for the compilers and libraries in case any issue arises during the compilation process. In general, it is recommended to use the provided pre-compiled binaries of XtalOpt.
